JavaScript

Variables

Variables in JS can be declared in 4 ways: 


                        //Using var
                        var variable;
                        //Using let
                        let variable;
                        //Using const
                        const variable;
                        //Using nothing
                        x = 5;

As for let, these rules follow:

  • • Cannot be redeclared
  • • Must be declared before use
  • • Has block scope

As for const, these rules follow:

  • • Cannot be redeclared
  • • Cannot be reassigned
  • • Has block scope

You can also utilize hoisting with variables, eg. say x = 5 and declare the variable
later down the code, as JS removes all declariations to the top of the current
scope (of script or function);

Arrays

An array in JS can be declared by: 


            const names = ['Steve', 'Brian', 'John'];
Array methods (some of them): 

            pop(); //removes last element from an array
            push(); //adds element to array
            shift(); //removes first index and shifts indexes
            length(); //length of array
            concat(arrToMerge); //merges two Arrays
            splice(); //adds new items to an array
            slice(); //slices out an item of an array
            map(); //creates a new array by calling a func. for each element
            forEach(); //loops through array

Callback Functions:

Function passed as an argument to another function.
Allows a function to call another function.

Example:


            const display(string) = () => {
                document.getElementById('textbox').innerHTML = string;
            }

            function hello(text1, text2, callback) {
                let string = text1 + text2;
                callback(string);
            }

            hello('John', 'Doe', display);

Arguably, callback functions get a bit more interresting when dealing with asynchronous code.
This also allowed me to showcase two ways of writing a JS function, either with function or
ES6 arrow functions.

Datatypes

JS variables can contain different types of datatypes, some examples: 


                        //String
                        let variable = 'hello';
                        //Integers
                        let variable = 11;
                        //Boolean
                        const variable = true;
                        //Objects
                        const person = {
                            name: 'John',
                            age: '25'
                        };
                        //Arrays
                        const numbs = [1, 2]

Datatypes are important. If they didn't exist, the computer/compiler would not know
how to interpret adding a string and a number together, eg. it should treat the number
as a string.
In JS a variable without a value has the value undefined.
Datatypes in JS are also dynamic.

Fetch

Fetch is used to make HTTP requests, both client and serverside, for Example
node-fetch in Express. Example on fetch below: 


            fetch('some uri')
            .then(res => res.json) //converts to json via. res.text() and JSON.parse()
            .then(data => {
                //do something with data
            });

Fetch supports all the other HTTP requests as well and a wide array of
options, all related to HTTP requests.
These options can be modified by passing in the options object.

Loops

In JS, we use loops to iterate over arrays, or for example
while loops to run based on a condition.

A traditional for loop for an array


            const numbers = [1, 2, 3, 4, 5];
            
            for (let i = 0: i < numbers.length; i++) {
                console.log(numbers[i]);
            }

However, a more functional approach is prefered, as per functional programming
as this produces simpler, safer and more readable code. Also immuteability
makes our objects more concise and less error-prone.

Examples: 


            const numbers = [1, 2, 3, 4, 5];
            //Do the same as the for loop
            numbers.forEach(element => console.log(element));
            
            //pass a function to map
            const map = numbers.map(number => number * 2);

            //filter results
            const res = numbers.filter(number => number < 2);
