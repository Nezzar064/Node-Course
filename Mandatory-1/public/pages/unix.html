CLI Cheatsheet

Cheat Sheet for useful/popular CLI commands, for use in prefered terminal. 


        mv: 
            Moves files/directories. Takes element and path as argument
        pwd:
            Prints working directory
        rm
            Removes file. Takes element to remove as argument
        rm -rf 
            Removes directories. Takes directory as argument
        echo
            Display text/string supplied as argument
        whoami
            Displays user using the terminal
        ipconfig / ifconfig
            Supplies info about network
        touch
            Create file
        mkdir
            Create directory
        cd
            Go to directory, takes path as argument
        cp:
            Copy, takes element path and where to copy to as argument
        cat
            Reads files
        ls
            Outputs directories/files in working directory
        ls -a 
            Outputs hidden directories/files in working directory
        cat > {filename}
            writes text into 
            the specified file
        echo 'line to write' > demo.txt
            Writes a single line to a file
        curl
            For HTTP requests
        tail 
            Useful for reading logs etc.
